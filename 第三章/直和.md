# 直和（Direct Sum）

## 獨立子空間

### 定義

考慮 $V$ 的子空間 $W_1, W_2, \ldots, W_k$，若

$$W_i \cap \sum_{j \neq i} W_j = \{0\}, \quad \forall i = 1, 2, \ldots, k$$

則稱 $W_1, W_2, \ldots, W_k$ 為**獨立子空間**。

### 等價定義

$$W_1, \ldots, W_k \text{ 獨立}$$
$$\Longleftrightarrow \forall w_i \in W_i:\; w_1 + w_2 + \cdots + w_k = 0 \Rightarrow w_1 = \cdots = w_k = 0$$
$$\Longleftrightarrow \dim(W_1 + \cdots + W_k) = \dim(W_1) + \cdots + \dim(W_k)$$

### 兩兩獨立 vs 獨立

- 獨立 $\Rightarrow$ 兩兩獨立（$W_i \cap W_j = \{0\}$，$\forall i \neq j$）
- **反之不成立**（$k \geq 3$ 時）

反例：$V = R^3$，$W_1 = x$-軸，$W_2 = y$-軸，$W_3$：直線 $x = y$（兩兩獨立但不獨立）

## 直和定義

若 $V = W_1 + W_2 + \cdots + W_k$ 且 $W_1, \ldots, W_k$ 為獨立子空間，稱

$$V = W_1 \oplus W_2 \oplus \cdots \oplus W_k$$

## 重要例子

### 對稱與斜對稱矩陣的直和

設 $V = F^{n \times n}$，
$$W_1 = \{A \mid A^T = A\}（對稱矩陣），\quad W_2 = \{A \mid A^T = -A\}（斜對稱矩陣）$$

$$\boxed{V = W_1 \oplus W_2}$$

分解公式：$A = \underbrace{\dfrac{A + A^T}{2}}_{\in W_1} + \underbrace{\dfrac{A - A^T}{2}}_{\in W_2}$

> **任何方陣 $=$ 對稱矩陣 $+$ 斜對稱矩陣**

### 奇函數與偶函數的直和

設 $V = \{f \mid f: R \to R\}$，
$$W_1 = \{f \mid f(-x) = f(x)\}（偶函數），\quad W_2 = \{f \mid f(-x) = -f(x)\}（奇函數）$$

$$V = W_1 \oplus W_2$$

分解：$f(x) = \underbrace{\dfrac{f(x)+f(-x)}{2}}_{\text{偶}} + \underbrace{\dfrac{f(x)-f(-x)}{2}}_{\text{奇}}$

> **任何函數 $=$ 偶函數 $+$ 奇函數**

## 與 $\dim$ 的關係

$$V = W_1 \oplus W_2 \Longleftrightarrow V = W_1 + W_2 \text{ 且 } W_1 \cap W_2 = \{0\}$$

## 經典考題

- 證明 $V = W_1 \oplus W_2$：需證 (1) $V = W_1 + W_2$（每個向量可分解），(2) $W_1 \cap W_2 = \{0\}$
- 判斷子空間是否獨立
