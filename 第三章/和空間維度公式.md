# 和空間的維度公式

## 定理

考慮有限維向量空間 $V$ 的子空間 $W_1, W_2$：

$$\boxed{\dim(W_1 + W_2) = \dim(W_1) + \dim(W_2) - \dim(W_1 \cap W_2)}$$

## 推論

$$W_1 \cap W_2 = \{0\} \Longleftrightarrow \dim(W_1 + W_2) = \dim(W_1) + \dim(W_2)$$

此時 $W_1 + W_2 = W_1 \oplus W_2$（直和）。

## 證明思路

令 $\{v_1, \ldots, v_k\}$ 為 $W_1 \cap W_2$ 的基底，

分別擴充為 $W_1$ 和 $W_2$ 的基底：
$$S_1 = \{v_1, \ldots, v_k, x_1, \ldots, x_m\},\quad S_2 = \{v_1, \ldots, v_k, y_1, \ldots, y_n\}$$

則 $\beta = \{v_1, \ldots, v_k, x_1, \ldots, x_m, y_1, \ldots, y_n\}$ 為 $W_1 + W_2$ 的基底，

$$\dim(W_1 + W_2) = k + m + n = (k+m) + (k+n) - k = \dim(W_1) + \dim(W_2) - \dim(W_1 \cap W_2)$$

## 注意

**三個子空間的容斥公式一般不成立**：
$$\dim(S+T+U) \neq \dim(S)+\dim(T)+\dim(U)-\dim(S\cap T)-\dim(S\cap U)-\dim(T\cap U)+\dim(S\cap T\cap U)$$

反例：$V=R^3$，$S=\text{span}\{(1,0,0)\}$，$T=\text{span}\{(0,1,0)\}$，$U=\text{span}\{(1,1,0)\}$。

## 解題技巧

**策略一（直接）**：
$$W_1 + W_2 = \text{span}(S_1 \cup S_2)$$
列矩陣做列運算，rank 即為 $\dim(W_1 + W_2)$，再反推 $\dim(W_1 \cap W_2)$。

**策略二（反推）**：
先求 $\dim(W_1)$，$\dim(W_2)$，$\dim(W_1+W_2)$，用公式求 $\dim(W_1 \cap W_2)$。

## 經典考題

- 給定 $W_1, W_2$ 的生成集，求 $\dim(W_1 \cap W_2)$
- 給定 $X_1 \neq X_2$，$\dim(X_1) = \dim(X_2) = 2$，$X_1 \cap X_2 \neq \{0\}$，求 $\dim(X_1+X_2)$ → 答案 3
