# 基底與維度（Basis & Dimension）

## 定義

向量空間 $V$ 的一個子集 $S$ 為 $V$ 的一組**基底**（basis），若：
1. $\text{span}(S) = V$（生成 $V$）
2. $S$ 為線性獨立集

基底中向量的個數稱為 $V$ 的**維度**，記成 $\dim(V)$。

## 性質

1. **任何兩組基底元素個數相同**（故維度定義合法）
2. $\dim(\{0\}) = 0$，空集合是零空間的唯一基底
3. $S$ 是基底 $\Leftrightarrow$ $S$ 是 $V$ 的**最小生成集**
4. $S$ 是基底 $\Leftrightarrow$ $S$ 是 $V$ 的**最大獨立集**
5. 若 $S$ 是基底，則 $V$ 中每個向量對 $S$ 有**唯一**線性表示

## 常見空間的標準基底（Standard Basis）

### 歐式空間 $F^n$

$$\{e_1=(1,0,\ldots,0),\; e_2=(0,1,\ldots,0),\; \ldots,\; e_n=(0,0,\ldots,1)\}$$
$$\dim(F^n) = n$$

| 空間 | $\dim$ |
|------|--------|
| $R^n/R$ 或 $C^n/C$ | $n$ |
| $C^n/R$ | $2n$ |

### 矩陣空間 $M_{m \times n}(F)$

$$\{E_{ij} \mid 1 \leq i \leq m,\; 1 \leq j \leq n\}，\text{ 其中 } E_{ij} \text{ 只有 } (i,j) \text{ 位置為 } 1$$
$$\dim(M_{m \times n}(F)) = mn$$

### 多項式空間

$$P_n[x]：\{1, x, x^2, \ldots, x^n\}，\quad \dim = n+1$$
$$P[x]：\{1, x, x^2, \ldots\}，\quad \dim = \infty$$

## 判斷基底的方法

對 $V$ 中的 $k$ 個向量 $\{v_1, \ldots, v_k\}$，若 $\dim(V) = k$，則：
- 只需驗**線性獨立**（自動生成）
- 或只需驗**生成 $V$**（自動獨立）

**計算技巧**：排成矩陣做列運算，rank $=$ 向量個數 → 獨立（是基底）。

## 擴張定理與緊縮定理

- **擴張**：獨立集可擴充為基底
- **緊縮**：生成集可縮減為基底

## 經典考題

- 判斷給定向量集是否為 $R^3$ 或 $P_2$ 的基底
- 找出子空間的一組基底並求維度
- 給定部分基底，找第三個向量使其成為完整基底（即 $\det \neq 0$）
