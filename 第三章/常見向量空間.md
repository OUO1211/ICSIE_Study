# 常見向量空間

## 定義

### 1. $n$ 維歐式空間（Euclidean space）

$$F^n = \{(x_1, x_2, \ldots, x_n) \mid x_i \in F\}$$

- 向量加法：$(x_1,\ldots,x_n) + (y_1,\ldots,y_n) = (x_1+y_1,\ldots,x_n+y_n)$
- 純量積：$\alpha (x_1,\ldots,x_n) = (\alpha x_1,\ldots,\alpha x_n)$
- 零向量：$(0,0,\ldots,0)$

| 空間 $V/F$ | 是向量空間？ | $\dim$ |
|------------|-------------|--------|
| $R^n/R$ | 是 | $n$ |
| $C^n/C$ | 是 | $n$ |
| $C^n/R$ | 是 | $2n$ |
| $R^n/C$ | **否** | $\times$ |

### 2. 矩陣空間（matrix space）

$$M_{m \times n}(F) = R^{m \times n} \text{ 或 } C^{m \times n}$$

- 加法 = 矩陣加法；純量積 = 矩陣純量積
- 零向量：零矩陣 $O$

### 3. 函數空間（function space）

$$V = \{f \mid f: D[a,b] \to F\}$$

- 零向量：零函數 $f(x) = 0$

#### 特殊子類

| 名稱 | 符號 | 說明 |
|------|------|------|
| 多項式空間 | $P(F) = F[x]$ | 所有多項式 |
| $n$ 次多項式空間 | $P_n(F) = F_n[x]$ | $\deg f \leq n$ 或 $f=0$ |
| 連續函數空間 | $C[a,b]$ | $[a,b]$ 上連續函數 |

> **注意**：有些書定義 $P_n[x]$ 收集 $\deg f < n$ 的多項式，$\dim = n$；另一種是 $\deg f \leq n$，$\dim = n+1$，注意題目定義。

## 經典考題

- 判斷 $(R^n/C)$ 是否為向量空間：否，因純量積 $\alpha x \notin R^n$ when $\alpha \in C$。
- 判斷 $C^2/R$ 的維度：$\dim = 4$，基底 $\{(1,0),(i,0),(0,1),(0,i)\}$。
