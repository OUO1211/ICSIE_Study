# 線性組合與生成空間

## 定義

### 線性組合（Linear Combination）

設向量 $v_1, v_2, \ldots, v_k \in V$，純量 $a_1, a_2, \ldots, a_k \in F$，則

$$v = a_1 v_1 + a_2 v_2 + \cdots + a_k v_k = \sum_{i=1}^k a_i v_i$$

稱為 $v_1, \ldots, v_k$ 的一個**線性組合**。

### 生成空間（Span）

集合 $S \subseteq V$ 的**所有有限線性組合**構成的集合，稱為 $S$ 的**生成空間**：

$$\text{span}(S) = \left\{ \sum_{i=1}^k a_i v_i \;\middle|\; v_i \in S,\; a_i \in F,\; k \in \mathbb{N} \right\}$$

若 $\text{span}(S) = V$，稱 $S$ 為 $V$ 的一個**生成集**（spanning set / generating set）。

> **特例**：$\text{span}(\emptyset) = \{0\}$

## 性質

1. $S \subseteq \text{span}(S)$
2. $S_1 \subseteq S_2 \Rightarrow \text{span}(S_1) \subseteq \text{span}(S_2)$
3. $\text{span}(S_1) \cup \text{span}(S_2) \subseteq \text{span}(S_1 \cup S_2)$
4. $\text{span}(S_1) \cap \text{span}(S_2) \supseteq \text{span}(S_1 \cap S_2)$
5. **$\text{span}(S)$ 是包含 $S$ 的最小子空間**（重要！）
6. 若 $S \subseteq W$（子空間），則 $\text{span}(S) \subseteq W$

## 與四大子空間的關係

$$RS(A) = \text{span}\{A_{(1)}, A_{(2)}, \ldots, A_{(m)}\}$$
$$CS(A) = \text{span}\{A^{(1)}, A^{(2)}, \ldots, A^{(n)}\}$$

## 和空間與生成集

$$\text{span}(S_1) + \text{span}(S_2) = \text{span}(S_1 \cup S_2)$$

## 經典考題

判斷向量 $b$ 是否在 $\text{span}(S)$ 中 → 解線性方程組 $Ax = b$（$A$ 的列/行為 $S$ 中向量）：
- 有解 → $b \in \text{span}(S)$
- 無解 → $b \notin \text{span}(S)$
