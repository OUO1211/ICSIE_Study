# 四大子空間的基底求法

## 核心方法：列運算

對矩陣 $A$ 做列運算，化成列梯形矩陣（REF）$B$（$A \sim B$）：

| 子空間 | 求法 | 說明 |
|--------|------|------|
| $RS(A)$（列空間） | 取 $B$ 的**非零列** | 列運算保列空間 |
| $\ker(A)$（核空間/零空間） | 解 $Bx = 0$（自由變數解） | 列運算保核空間 |
| $CS(A)$（行空間） | 找 $B$ 的 pivot 所在**列數**，取回 $A$ 對應的**行向量** | 不能用 $B$ 的行！ |
| $L\ker(A)$（左核空間） | 對 $A^T$ 求列空間 / 或用 $CS(A)$ 的正交補 | |

> **關鍵**：$CS(A)$ 的基底必須取自**原矩陣 $A$** 的行向量（對應 pivot 列），不能用 $B$。

## 求 $\ker(A)$ 步驟（演算法）

**Step 1**：對 $A$ 做列運算化為 RREF（$B$）

**Step 2**：從 $Bx = 0$ 讀出約束：
$$\begin{cases} x_{p_1} = f(\text{自由變數}) \\ x_{p_2} = g(\text{自由變數}) \\ \vdots \end{cases}$$

**Step 3**：自由變數各取 $1$，其餘為 $0$，寫出基礎解向量

**Step 4**：$\ker(A) = \text{span}\{\text{基礎解向量}\}$

> 自由變數數目 $=$ nullity$(A)$ $=$ $n - \text{rank}(A)$

## 利用 LU 分解求 CS(A)

若 $A = LU$（$L$ 可逆，$U$ 為梯形），$\text{rank}(A) = r$：
- $CS(A)$ 的基底 $=$ $L$ 的**前 $r$ 個行向量**

## Sylvester 定理（Rank-Nullity Theorem）

$$\boxed{n = \text{rank}(A) + \text{nullity}(A)}$$

其中 $\text{nullity}(A) = \dim(\ker(A))$，$n$ 為 $A$ 的行數（columns）。

## 經典考題

給定矩陣 $A$，求：
1. $RS(A)$ 的基底 → 列運算 → 非零列
2. $\ker(A)$ 的基底 → 列運算 → 解 $Ax=0$
3. $CS(A)$ 的基底 → 找 pivot 位置 → 取原始 $A$ 的行向量
4. $L\ker(A) = N(A^T)$ 的基底 → 對 $A^T$ 做同樣流程
