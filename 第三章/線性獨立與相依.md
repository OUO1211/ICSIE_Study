# 線性獨立與線性相依

## 定義

考慮向量空間 $V$ 中的有限集合 $\{v_1, v_2, \ldots, v_k\}$：

$$\text{若存在不全為零的純量 } c_1, \ldots, c_k \text{ 使得 } c_1 v_1 + \cdots + c_k v_k = 0$$

則稱此集合為**線性相依集**（linearly dependent）；否則稱為**線性獨立集**（linearly independent）。

**等價定義**（獨立）：$c_1 v_1 + \cdots + c_k v_k = 0 \Rightarrow c_1 = c_2 = \cdots = c_k = 0$

## 性質

1. 集合非獨立即相依（二擇一）
2. 相依 ⟺ 其中某向量可表為其他向量的線性組合
3. $\{0\}$ 為相依集；含零向量的集合必相依
4. $\emptyset$ 為獨立集
5. $A \subseteq B$：若 $A$ 相依 $\Rightarrow$ $B$ 相依；若 $B$ 獨立 $\Rightarrow$ $A$ 獨立
6. 若 $\dim(V) = k$，則 $V$ 中超過 $k$ 個向量必相依
7. $\{v_1,\ldots,v_n\}$ 為 $R^n$ 的獨立集 $\Leftrightarrow$ $A=[v_1,\ldots,v_n]$ 可逆（$\det(A) \neq 0$）
8. 若 $\{v_1,\ldots,v_n\}$ 獨立且 $A$ 可逆，則 $\{Av_1,\ldots,Av_n\}$ 也獨立

> **體域的影響**：$\{(1,0),(i,0)\}$ 在複數體下相依，在實數體下獨立。

## 判斷方法

### 方法一：列運算（矩陣秩）

將向量排成矩陣的行（或列），做列運算：
- 無零列 → 獨立
- 有零列 → 相依

### 方法二：行列式(可逆)

判斷 $\{v_1,\ldots,v_n\}$ 是否獨立 $\rightarrow$ $c_1v_1+...+c_nv_n = 0$ 是否只有$c_1 =...=c_n有解$

$\rightarrow$ 化為kernal($Ax = 0$) $\rightarrow$ 只有0解(可逆) $\rightarrow$ A(方陣) det(A) $\neq$ 0  (或是用其他可逆判斷)  

$n$ 個 $n$ 維向量 $\{v_1,\ldots,v_n\}$：
$$\det[v_1, v_2, \ldots, v_n] \neq 0 \Leftrightarrow \text{獨立}$$

### 方法三：維度比較

若向量個數 $>$ $\dim(V)$，直接判定**相依**。

### 方法四：Wronskian（函數空間）

見 [[Wronskian]]。

### 方法五：定義法（代入 $0$，利用已知獨立集）

設 $\alpha_1 w_1 + \cdots + \alpha_k w_k = 0$，代入定義展開，
利用已知獨立集的係數必為零來推出 $\alpha_i = 0$。

## 與矩陣的關係

$$B = CA \Rightarrow \text{若 } A, C \text{ 均可逆，則 } B \text{ 的行向量獨立}$$

## 經典考題

- 已知 $\{u,v,w\}$ 獨立，判斷 $\{u+v, v+w, w+u\}$ 是否獨立
- 找使向量集線性相依的參數 $a$（令行列式 $= 0$）
