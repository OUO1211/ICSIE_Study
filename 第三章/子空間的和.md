# 子空間的和（Sum Space）

## 定義

考慮 $V$ 的子空間 $W_1, W_2, \ldots, W_k$，定義其**和空間**：

$$W_1 + W_2 + \cdots + W_k = \{v_1 + v_2 + \cdots + v_k \mid v_i \in W_i\}$$

## 性質

1. **$W_1 + W_2$ 是 $V$ 的子空間**（重要！）
2. $W_1 \cup W_2 \subseteq W_1 + W_2$
3. $W_1 + W_2$ 是**包含 $W_1 \cup W_2$ 的最小子空間**
4. $W_1 \cup W_2$ 不一定是子空間，但 $W_1 + W_2$ 必是子空間

## 和空間與生成集的關係

$$\text{若 } W_1 = \text{span}(S_1),\; W_2 = \text{span}(S_2)，\text{ 則}$$

$$\boxed{W_1 + W_2 = \text{span}(S_1 \cup S_2)}$$

**應用**：求 $W_1 + W_2$ 的生成集，只需對各子空間分別取生成集，再取聯集即可。

## 維度公式

$$\boxed{\dim(W_1 + W_2) = \dim(W_1) + \dim(W_2) - \dim(W_1 \cap W_2)}$$

推論：$\dim(W_1 + W_2) = \dim(W_1) + \dim(W_2) \Leftrightarrow W_1 \cap W_2 = \{0\}$

> **注意**：三個子空間的對應公式（容斥原理）一般**不成立**。

## 證明（$W_1+W_2$ 為子空間）

任取 $u = u_1 + u_2,\; v = v_1 + v_2 \in W_1 + W_2$，$a \in F$：

$$a u + v = (au_1 + v_1) + (au_2 + v_2)$$

因 $W_1, W_2$ 各自為子空間，$au_1 + v_1 \in W_1$，$au_2 + v_2 \in W_2$，故 $au + v \in W_1 + W_2$。

## 經典考題

給定 $W_1, W_2$ 的生成集，求 $\dim(W_1 + W_2)$ 與 $\dim(W_1 \cap W_2)$：
1. 用 $W_1 + W_2 = \text{span}(S_1 \cup S_2)$ 列矩陣做列運算求 rank
2. 再用維度公式反推 $\dim(W_1 \cap W_2)$
