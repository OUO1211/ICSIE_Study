# 四大基本子空間（Fundamental Subspaces）

## 定義

考慮 $m \times n$ 矩陣 $A$：

| 名稱                         | 符號                  | 定義                                                   |
| -------------------------- | ------------------- | ---------------------------------------------------- |
| **列空間**（row space）         | $RS(A)$             | $A$ 的列向量的所有線性組合 $= \{xA \mid x \in F^{1 \times m}\}$ |
| **行空間**（column space）      | $CS(A)$             | $A$ 的行向量的所有線性組合 $= \{Ax \mid x \in F^{n \times 1}\}$ |
| **核空間**（kernel/null space） | $\ker(A) = N(A)$    | $\{x_{n \times 1} \mid Ax = 0\}$（齊次方程的解集合）           |
| **左核空間**（left kernel）      | $L\ker(A) = N(A^T)$ | $\{y_{1 \times m} \mid yA = 0\}$                     |

## 子空間所在的空間

$$RS(A) \subseteq F^{1 \times n}, \quad CS(A) \subseteq F^{m \times 1}$$
$$\ker(A) \subseteq F^{n \times 1}, \quad L\ker(A) \subseteq F^{1 \times m}$$

## 維度關係（Sylvester 定理）

$$\boxed{n = \dim(\ker(A)) + \dim(CS(A)) = \text{nullity}(A) + \text{rank}(A)}$$
$$m = \dim(L\ker(A)) + \text{rank}(A)$$

## 可逆矩陣的等價條件（與四大子空間相關）

對 $n$ 階方陣 $A$，以下等價：
1. $\ker(A) = \{0\}$（核空間只含零向量）
2. 行向量為線性獨立
3. 行向量生成 $F^{n \times 1}$，即 $CS(A) = F^{n \times 1}$
4. $L\ker(A) = \{0\}$
5. 列向量為線性獨立，$RS(A) = F^{1 \times n}$

## 求基底的方法

對 $A$ 做列運算得列梯形矩陣 $B$：

| 子空間        | 求法                             |
| ---------- | ------------------------------ |
| $RS(A)$    | 取 $B$ 的非零列                     |
| $\ker(A)$  | 解 $Bx = 0$，等同解 $Ax = 0$        |
| $CS(A)$    | 找 $B$ 的 pivot 列，對應回 $A$ 的行向量   |
| $L\ker(A)$ | 對 $A^T$ 做列運算，或利用 $CS(A)^\perp$ |
