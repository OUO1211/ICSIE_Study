# 矩陣的秩（Rank）

## 定義

對矩陣 $A$：

- **列秩**（row rank）：$rr(A) = \dim(RS(A))$
- **行秩**（column rank）：$cr(A) = \dim(CS(A))$

## 核心定理：列秩 = 行秩

$$\boxed{rr(A) = cr(A) = \text{rank}(A)}$$

## Sylvester 定理（Rank-Nullity Theorem）

設 $A$ 為 $m \times n$ 矩陣：

$$\boxed{n = \text{rank}(A) + \text{nullity}(A) = \dim(CS(A)) + \dim(\ker(A))}$$
$$m = \text{rank}(A) + \dim(L\ker(A))$$

> **nullity**$(A) = \dim(\ker(A))$ = 自由變數個數

## 重要性質

設 $A$ 為 $m \times n$ 矩陣：

| 性質 | 公式 | 備注 |
|------|------|------|
| 轉置不改秩 | $\text{rank}(A) = \text{rank}(A^T)$ | 重要 |
| 上界 | $\text{rank}(A) \leq \min\{m, n\}$ | |
| 運算不改秩 | $A \sim B \Rightarrow \text{rank}(A) = \text{rank}(B)$ | 列/行運算 |
| 乘積上界 | $\text{rank}(AB) \leq \min\{\text{rank}(A), \text{rank}(B)\}$ | 很重要 |
| 可逆矩陣不改秩 | $P, Q$ 可逆時，$\text{rank}(PA) = \text{rank}(AQ) = \text{rank}(A)$ | 重要 |
| 加法上界 | $\text{rank}(A+B) \leq \text{rank}(A) + \text{rank}(B)$ | |
| 三角不等式 | $|\text{rank}(A) - \text{rank}(B)| \leq \text{rank}(A+B)$ | |

## 進階：Sylvester 不等式

$$\text{rank}(AB) \geq \text{rank}(A) + \text{rank}(B) - n$$

其中 $A: m \times n$，$B: n \times p$。

## 特殊情況

### $AB = O$ 時

$$CS(B) \subseteq \ker(A) \Rightarrow \text{rank}(A) + \text{rank}(B) \leq n$$

### $A = xy^T$（rank-1 矩陣）

設 $x \in R^m$，$y \in R^n$，均非零，則：
$$\text{rank}(xy^T) = 1, \quad CS(xy^T) = \text{span}\{x\}$$
$$\ker(xy^T) = \{v \mid y^T v = 0\}, \quad \dim(\ker(xy^T)) = n-1$$

## 秩的計算

1. 列運算到梯形，非零列數 $=$ rank
2. 行運算到梯形，非零行數 $=$ rank
3. 利用 Sylvester 定理反推
