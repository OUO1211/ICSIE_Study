## 定義

> **對稱矩陣 (symmetric matrix)**：$A^T = A$，即 $a_{ij} = a_{ji}$
>
> **斜對稱矩陣 (skew-symmetric matrix)**：$A^T = -A$，即 $a_{ij} = -a_{ji}$

## 性質

- 任何方陣 $A$ 可分解為對稱與斜對稱部分：

$$A = \underbrace{\frac{A + A^T}{2}}_{\text{對稱}} + \underbrace{\frac{A - A^T}{2}}_{\text{斜對稱}}$$

- $AA^T$ 和 $A^T A$ 均為對稱矩陣
- 斜對稱矩陣的對角項必為 $0$

## 經典考題

〔104 政大資科〕**For any square matrix $A$, show that $B = \frac{A+A^T}{2}$ is symmetric.**

**解：** $B^T = \left(\frac{A + A^T}{2}\right)^T = \frac{A^T + A}{2} = B$ ✓

〔106 交大資工〕**Let $B = A - A^T$. Show that $B$ is skew-symmetric.**

**解：** $B^T = A^T - A = -B$ ✓
