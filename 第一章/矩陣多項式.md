## 定義

> 設 $f(x) = c_n x^n + c_{n-1} x^{n-1} + \cdots + c_1 x + c_0$ 為多項式，$A$ 為 $n$ 階方陣，則：
>
> $$f(A) = c_n A^n + c_{n-1} A^{n-1} + \cdots + c_1 A + c_0 I$$

## 應用

利用多項式分解求反矩陣或冪次：

**例：若 $A^2 = A$（冪等），利用 $f(x) = (x+1)^n$：**

$$f(A) = (A + I)^n = \sum_{i=0}^{n} \binom{n}{i} A^i = I + (2^n - 1)A$$

（因為 $A^k = A$，$\forall k \geq 1$）〔107 台大資工〕

**例：冪零矩陣 $A^k = O$ 時：**

$$f(x) = 1 - x^k = (1-x)(1 + x + \cdots + x^{k-1})$$

$$\implies (I-A)^{-1} = I + A + A^2 + \cdots + A^{k-1}$$
