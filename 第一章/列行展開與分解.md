## 定義

設 $A_{(i)}$ 為矩陣 $A$ 的第 $i$ 個列向量，$y = [y_1 \ y_2 \ \cdots \ y_m]$ 為列向量。

### 列展開 (Row Expansion)

$$yA = y_1 A_{(1)} + y_2 A_{(2)} + \cdots + y_m A_{(m)}$$

即 $y$ 乘以 $A$ 等於 $y$ 的各分量對 $A$ 的各列向量的線性組合。

### 列切割 (Row Slicing)

$$AB = C \implies A_{(i)} B = C_{(i)}, \quad \forall i = 1, 2, \ldots, m$$

### 行切割 (Column Slicing)

設 $A^{(j)}$ 為 $A$ 的第 $j$ 行向量，則：

$$AB = C \implies A B^{(j)} = C^{(j)}, \quad \forall j = 1, 2, \ldots, p$$

## 應用

- 矩陣方程式 $AX = B$ 可按行分解為 $A x^{(j)} = b^{(j)}$
- 加速理解線性方程組的幾何意義
