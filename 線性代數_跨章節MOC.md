# 線性代數 跨章節連結 MOC

> 本 MOC 的目的：找出第一、二、三章間**概念繼承與相互依賴**的關係，
> 讓複習時能清楚看到「哪些工具從哪裡來、用在哪裡」。

---

## 總覽：三章的定位

```
第一章：矩陣
  → 定義運算工具（矩陣乘法、列運算、LU分解、RREF）

第二章：線性方程組與行列式
  → 使用工具解問題（解 Ax=b、det判斷可逆、Cramer法則）

第三章：向量空間
  → 抽象化與推廣（子空間、基底、維度、秩的深層意義）
```

## 一、工具對應關係：「工具在第一章，應用在第三章」

### 1-1 RREF 與列運算 → 求四大子空間基底

| 第一章（工具） | 第三章（應用） |
|--------------|--------------|
| [[RREF簡化列梯陣]] — 化矩陣到最簡形 | [[四大子空間的基底]] — 列運算求 $RS(A)$、$\ker(A)$、$CS(A)$ |
| [[基本列運算]] — 三型列運算 | 列運算保列空間（$RS(A)$ 不變） |
| [[利用列運算求反矩陣]] — $[A\|I] \to [I\|A^{-1}]$ | 利用 $[B'\|B] \to [I\|P]$ 求[[座標與基底變換\|座標變換矩陣]] |

> **關鍵事實**：列運算保 $RS(A)$ 和 $\ker(A)$，但**不保 $CS(A)$** — 求 $CS(A)$ 基底必須回到原矩陣 $A$ 的 pivot 行。

---

### 1-2 LU/PAQ 分解 → 秩與行空間

| 第一章（工具） | 第三章（應用） |
|--------------|--------------|
| [[LU分解]] — $A = LU$，$L$ 可逆 | $CS(A)$ 基底 $=$ $L$ 的前 $r$ 個行向量（$r = \text{rank}(A)$） |
| [[PAQ分解]] — $PAQ = \begin{bmatrix}I_r & O \\ O & O\end{bmatrix}$ | $r$ 就是 [[矩陣的秩\|rank$(A)$]]；$P$、$Q$ 可逆不改秩 |
| [[秩Rank]]（第一章版本） | [[矩陣的秩]]（第三章版本）— 嚴格定義 $rr(A) = cr(A)$，Sylvester 定理 |

---

### 1-3 對稱/斜對稱矩陣 → 直和分解

| 第一章（工具） | 第三章（應用） |
|--------------|--------------|
| [[對稱矩陣與斜對稱矩陣]] — 任意方陣 $= \frac{A+A^T}{2} + \frac{A-A^T}{2}$ | [[直和]] — $F^{n\times n} = W_1 \oplus W_2$（對稱 $\oplus$ 斜對稱） |

## 二、概念傳承：「第二章建立，第三章深化」

### 2-1 零空間（第二章）= ker(A)（第三章）

| 第二章 | 第三章 |
|--------|--------|
| [[齊次方程組與零空間]] — 解集 $\{x \mid Ax=0\}$ | [[四大基本子空間]] — $\ker(A)$ 是向量空間的子空間 |
| [[基本變數與自由變數]] — 自由變數個數 $= n - \text{rank}$ | [[矩陣的秩]] — nullity$(A) = n - \text{rank}(A)$（Sylvester） |
| [[解空間與解的結構]] — 特殊解 + 零空間 | [[子空間]] — 零空間是子空間；解集合 $x_0 + \ker(A)$ 是**仿射子空間** |

---

### 2-2 相容性判斷（第二章）→ 秩與線性系統（第三章）

| 第二章 | 第三章 |
|--------|--------|
| [[增廣矩陣與相容性]] — 有解 $\Leftrightarrow$ rank$([A\|b])$ = rank$(A)$ | [[秩與線性系統]] — $b \in CS(A)$ ↔ 有解（用行空間語言重述） |
| [[高斯消去法]]、[[高斯-喬登消去法]] | [[四大子空間的基底]] — RREF 是求基底的核心計算工具 |

---

### 2-3 可逆等價條件（第二章）→ 四大子空間（第三章）

第二章的「七條等價」在第三章獲得**空間語言的重新詮釋**：

| 第二章等價條件 | 第三章語言 |
|--------------|-----------|
| $Ax=0$ 只有零解 | $\ker(A) = \{0\}$，nullity$(A) = 0$ |
| $\text{rank}(A) = n$ | $\dim(CS(A)) = n$，$CS(A) = F^n$ |
| 行向量線性獨立 | $\ker(A) = \{0\}$（無自由變數） |
| $Ax = b$ 有唯一解 | $b \in CS(A)$ 且 $\ker(A) = \{0\}$ |

相關笔記：[[可逆矩陣等價條件]] ↔ [[四大基本子空間]]

---

### 2-4 行列式 → 線性獨立判斷（第三章）

| 第二章 | 第三章 |
|--------|--------|
| [[行列式進階性質]] — $\det(A) \neq 0 \Leftrightarrow A$ 可逆 | [[線性獨立與相依]] — $n$ 個 $n$ 維向量獨立 $\Leftrightarrow \det \neq 0$ |
| [[Laplace展開法]] | [[Wronskian]] — 也是行列式，用於判斷函數獨立性 |
| [[特殊矩陣的行列式]] | [[線性獨立與相依]] — 超過 $\dim(V)$ 個向量必相依 |

## 三、核心概念演化路線

### 秩（Rank）的演化

```
第一章：秩Rank
  → 以 RREF 非零列定義（操作性定義）
  → 與可逆性、PAQ分解掛鉤

第二章：rank 出現在
  → 相容性條件：rank([A|b]) = rank(A)
  → 基本/自由變數個數

第三章：矩陣的秩（完整理論）
  → 列秩 = 行秩（證明）
  → Sylvester 定理：n = rank + nullity
  → 秩的七大性質（含 Sylvester 不等式）
  → 與子空間維度全面整合
```

### 子空間的演化

```
第一章：
  → 列空間、行空間（隱性出現在 LU 分解、RREF）

第二章：
  → 零空間 ker(A)（齊次方程解集）
  → 解空間（仿射子空間）

第三章：
  → 子空間的嚴格公理定義
  → 四大子空間作為獨立研究對象
  → 子空間的基底、維度、和空間、直和
```

### 線性獨立的演化

```
第二章：
  → 行向量獨立 ↔ 可逆（七條等價之一）

第三章：
  → 獨立的完整定義與判斷（五種方法）
  → Wronskian（函數空間特有）
  → 獨立集與基底的關係
```

## 四、考研跨章節高頻題型

### 題型 A：「可逆性判斷」連結三章

$$A \text{ 可逆}$$
$$\Updownarrow$$
$$\text{（第一章）} \det(A) \neq 0, \; A = LU \text{ 分解存在}$$
$$\Updownarrow$$
$$\text{（第二章）} Ax = 0 \text{ 只有零解}, \; \text{rank}(A) = n$$
$$\Updownarrow$$
$$\text{（第三章）} \ker(A) = \{0\}, \; CS(A) = F^n, \; \text{行向量為基底}$$

**參考筆記**：[[可逆矩陣]] → [[可逆矩陣等價條件]] → [[四大基本子空間]]

---

### 題型 B：「給矩陣求四大子空間」（第一章工具 + 第三章定義）

---

### 題型 C：「rank + 解的個數」（第二章相容 + 第三章秩）

- $Ax = b$ 有解：[[增廣矩陣與相容性]] ↔ [[秩與線性系統]]
- 解唯一：rank$(A)$ = $n$（[[基本變數與自由變數]] ↔ nullity = 0）
- 無限多解：nullity$(A) > 0$（[[解空間與解的結構]] ↔ [[矩陣的秩]]）
