### (9) RS(AB) ⊆ RS(B)；CS(AB) ⊆ CS(A)

**【證明】** 【104 中興資科】

設 **y** ∈ RS(AB)，即存在 **x** 使 **y** = **x**AB，則 **y** = (**x**A)B ∈ RS(B)，故 RS(AB) ⊆ RS(B)。

設 **y** ∈ CS(AB)，即存在 **x** 使 **y** = AB**x**，則 **y** = A(B**x**) ∈ CS(A)，故 CS(AB) ⊆ CS(A)。

---

### (10) L ker(A) ⊆ L ker(AB)；ker(B) ⊆ ker(AB)

**【證明】**

任取 **x** ∈ L ker(A)，則 **x**A = **0**，∴ (**x**A)B = **0**，∴ **x** ∈ L ker(AB)，

∴ L ker(A) ⊆ L ker(AB)。

任取 **x** ∈ ker(B)，則 B**x** = **0**，∴ AB**x** = A**0** = **0**，∴ **x** ∈ ker(AB)，

∴ ker(B) ⊆ ker(AB)。

---

### (11) 若 AB = O，則 CS(B) ⊆ ker(A)；RS(A) ⊆ L ker(B)

**【證明】** 【90.92 交大資工、93.100 成大統計、102 嘉義應數、108 中山應數】

設 **y** ∈ CS(B)，即存在 **x** 使 **y** = B**x**，則 A**y** = AB**x** = O**x** = **0**，故 **y** ∈ ker(A)，

∴ CS(B) ⊆ ker(A)。

設 **y** ∈ RS(A)，即存在 **x** 使 **y** = **x**A，則 **y**B = **x**AB = **x**O = **0**，∴ **y** ∈ L ker(B)，

故 RS(A) ⊆ L ker(B)。

### (12) 與線性系統 Ax = b 的關係：Ax = b 有解若且唯若 b ∈ CS(A)

**【證明】**

Ax = b 有解

⟺ 存在 x = [c₁, ..., cₙ]^T，使 Ax = b，

⟺ 存在 c₁, ..., cₙ，使 c₁A^(1) + ... + cₙA^(n) = b，

⟺ b ∈ CS(A)，

其中，A^(1), ..., A^(n) 為 A 的行向量。
