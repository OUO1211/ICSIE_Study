# 四大子空間的性質

## Note（名詞說明）

(1) CS(A) 也被記成 Range(A)、R(A)、或 Im(A)。

(2) 因 RS(A) 與 CS(A^T) 同構（第四章會介紹），故也有將 A 的列空間定義成 CS(A^T)。

(3) ker(A) 也被記成 Nullspace(A)、Null(A)、或 N(A)。

(4) 因 L ker(A) 與 ker(A^T) 同構，故也有將 A 的左核空間定義成 ker(A^T)。

(5) RS(A) 為 R^(1×n) 的子空間；CS(A) 為 R^(m×1) 的子空間。

---

## 證明

### (5) RS(A) 為 R^(1×n) 的子空間；CS(A) 為 R^(m×1) 的子空間

**【證明】**

明顯可得 **0** ∈ RS(A)，

設 **x**A, **y**A ∈ RS(A)，for some **x**, **y** ∈ R^(1×m)，設 α ∈ R，

則 α(**x**A) + **y**A = (α**x** + **y**)A ∈ RS(A)，

∴ RS(A) 為 R^(1×n) 的子空間。

CS(A) 為 R^(m×1) 的子空間，證法類似上述。

---

### (6) ker(A) 為 R^(n×1) 的子空間；L ker(A) 為 R^(1×m) 的子空間

**【證明】**

明顯可得 **0** ∈ ker(A)，

設 **x**, **y** ∈ ker(A)，即 A**x** = **0**，A**y** = **0**，設 α ∈ R，

則 A(α**x** + **y**) = αA**x** + A**y** = α**0** + **0** = **0**，∴ α**x** + **y** ∈ ker(A)，

∴ ker(A) 為 R^(n×1) 的子空間。

L ker(A) 為 R^(1×m) 的子空間，證法類似上述。

---

### (7) 列運算不改列空間與核空間，即若 A ~(r) B，則 RS(A) = RS(B) 且 ker(A) = ker(B)

**【證明】**

設 A 列等價於 B，即存在可逆矩陣 P 使 B = PA，

設 **y** ∈ RS(A)，即存在 **x** ∈ F^(1×m)，使 **y** = **x**A，

則 **y** = **x**(P⁻¹B) = (**x**P⁻¹)B ∈ RS(B)，故 RS(A) ⊆ RS(B)。

設 **y** ∈ RS(B)，即存在 **x** ∈ F^(1×m)，使 **y** = **x**B，

則 **y** = **x**(PA) = (**x**P)A ∈ RS(A)，故 RS(B) ⊆ RS(A)。

故 RS(A) = RS(B)。

類似證法可得 ker(A) = ker(B)。
