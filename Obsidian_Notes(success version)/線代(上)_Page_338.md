> **дҫӢйЎҢ 7**
>
> Let $\{v_1, v_2, v_3, v_4\}$ be a basis of a vector space $V$ over a field $F$. Determine if the following set is a basis of $V$. Justify your answer.
>
> (1) (5%) $\{v_1, v_1 + v_2, v_1 + v_2 + v_3, v_1 + v_2 + v_3 + v_4\}$гҖӮ гҖҗ98 дёӯжӯЈиіҮе·ҘгҖ‘
>
> (2) (5%) $\{v_1 + v_2, v_2 + v_3, v_3 + v_4, v_4 + v_1\}$гҖӮ гҖҗ83 еҸ°еӨ§иіҮе·Ҙ йЎҢзө„пјҢ96 жҲҗеӨ§ ж•ёеӯёгҖ‘

**и§Ј**

$\because \{v_1, v_2, v_3, v_4\}$ зӮә $V$ зҡ„еҹәеә•пјҢж•… $\dim(V) = 4$гҖӮ

(1) YesгҖӮ

е…ҲиӯүжҳҺжӯӨзӮәз·ҡжҖ§зҚЁз«ӢйӣҶпјҡ

иЁӯ $\alpha_1 v_1 + \alpha_2(v_1 + v_2) + \alpha_3(v_1 + v_2 + v_3) + \alpha_4(v_1 + v_2 + v_3 + v_4) = 0$пјҢ

еүҮ $(\alpha_1 + \alpha_2 + \alpha_3 + \alpha_4)v_1 + (\alpha_2 + \alpha_3 + \alpha_4)v_2 + (\alpha_3 + \alpha_4)v_3 + \alpha_4 v_4 = 0$пјҢ

иҖҢ $\{v_1, v_2, v_3, v_4\}$ зӮәз·ҡжҖ§зҚЁз«ӢпјҢж•… $\begin{cases} \alpha_1 + \alpha_2 + \alpha_3 + \alpha_4 = 0 \\ \alpha_2 + \alpha_3 + \alpha_4 = 0 \\ \alpha_3 + \alpha_4 = 0 \\ \alpha_4 = 0 \end{cases}$пјҢ

$\therefore \alpha_1 = \alpha_2 = \alpha_3 = \alpha_4 = 0$пјҢ

$\therefore \{v_1, v_1+v_2, v_1+v_2+v_3, v_1+v_2+v_3+v_4\}$ зӮәз·ҡжҖ§зҚЁз«ӢйӣҶпјҢ

еҸҲеӣ зӮә $\dim V = 4$ еҖӢеҗ‘йҮҸпјҢж•…еҸҜзӮә $V$ зҡ„дёҖзө„еҹәеә•гҖӮ

(2) NoгҖӮ

$\because (v_1 + v_2) - (v_2 + v_3) + (v_3 + v_4) - (v_4 + v_1) = 0$пјҢ

$\therefore \{v_1 + v_2, v_2 + v_3, v_3 + v_4, v_4 + v_1\}$ зӮәз·ҡжҖ§зӣёдҫқпјҢж•…дёҚзӮәеҹәеә•гҖӮ
