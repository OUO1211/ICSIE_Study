> **例 9**
>
> Let square matrix $A$ satisfy $A^k = 0$, for some integer $k$（即為某零 nilpotent 方陣）. Prove that $I + A$ is invertible.
>
> 【105 台大資工】

**解** 設 $(I + A)x = 0$，

則 $Ax = -x$，

$\therefore A^2 x = -Ax = (-1)^2 x$，

$\vdots$

$\therefore A^k x = (-1)^k x$，

$\therefore 0x = (-1)^k x$，

$\therefore 0 = x$，$\therefore I + A$ 為可逆矩陣。
